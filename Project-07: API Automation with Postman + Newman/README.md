# Project-07_API_Automation_with_Postman_Newman

## Project Overview

This project demonstrates API automation testing using Postman and Newman. The objective is to automate CRUD API test execution, validate responses through JavaScript assertions, and generate automated execution reports from the command line.

## Tools Used

* Postman
* Newman
* Node.js
* JavaScript (Postman Test Scripts)
* Google Chrome
* Windows 10

## API Source

ReqRes API (https://reqres.in)

## Scope of Automation

The following CRUD operations were automated:

* Create User (POST)
* Get User (GET)
* Update User (PUT)
* Partial Update User (PATCH)
* Delete User (DELETE)

## Automated Validations

* HTTP status code validation
* Response body validation
* JSON field validation
* Token and ID validation
* Timestamp validation
* Response time validation
* Negative API response validation

## Newman Execution

Postman collections were executed using Newman from the command line, and automated execution reports were generated for analysis and documentation.

## Project Deliverables

* Automated Postman Collection (.json)
* Newman HTML Report (.html)
* Newman Console Execution Report (.txt)
* API Automation Test Cases (.xlsx)
* API Automation Execution Report (.pdf/.docx)

## Key Learning Outcomes

* API automation using Postman
* Writing JavaScript assertions with pm.test()
* Running collections through Newman
* Generating automated execution reports
* Validating API responses programmatically
* Basic command-line automation workflow
