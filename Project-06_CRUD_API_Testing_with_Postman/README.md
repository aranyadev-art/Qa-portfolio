# Project-06_CRUD_API_Testing_with_Postman

## Project Overview

This project demonstrates manual CRUD API testing using Postman on the ReqRes API. The objective is to validate Create, Read, Update, and Delete operations along with status code verification, request validation, response validation, and API error handling.

## Tools Used

* Postman
* Microsoft Excel
* Google Chrome
* Windows 10

## API Source

ReqRes API (https://reqres.in)

## CRUD Operations Covered

* Create User (POST)
* Get User (GET)
* Update User (PUT)
* Partial Update User (PATCH)
* Delete User (DELETE)

## Test Scenarios Covered

* Valid Create Request
* Create with Empty Fields
* Get Existing User
* Get Non-Existing User
* Full Update Validation
* Partial Update Validation
* Delete Existing User
* Delete Invalid User
* Wrong HTTP Method
* Invalid Endpoint
* Response Status Code Validation
* Response Body Validation
* Response Time Validation

## Expected Deliverables

* CRUD API Test Cases (.xlsx)
* CRUD API Bug Report (.xlsx)
* CRUD API Test Execution Report (.docx/.pdf)
* Postman Collection (.json)

## Key Learning Outcomes

* REST API CRUD testing
* POST, GET, PUT, PATCH, and DELETE methods
* Status code validation (200, 201, 204, 400, 401, 404, 405)
* JSON request and response validation
* API error handling
* Bug reporting and execution reporting
