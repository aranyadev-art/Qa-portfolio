# Amazon Shopping Cart Manual Testing

## Project Overview

This project contains manual testing documentation for the Amazon Shopping Cart module on Amazon.in.

## Scope

The following cart features were tested:

* Add to Cart
* Remove from Cart
* Increase Quantity
* Decrease Quantity
* Cart Subtotal Calculation
* Multiple Product Handling
* Save for Later
* Product Navigation from Cart
* Cart Persistence after Refresh
* Empty Cart Behavior
* Product Sharing Functionality
* Recommended Product Interaction
